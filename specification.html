Specification

Read the most up to date version of the Solid Specification.

Anyone can contribute to the Solid Specification by submitting issues and pull requests. Before contributing please read the Code of Conduct and join the W3C Solid Community Group.

It is suggested that you present your proposals at the relevant calls and communication channels, which include the following weekly recurring meetings:

  • Mondays at 1600 CEST Authentication Panel on this line
  • Mondays at 2130 CEST Interoperability Panel on this line
  • Wednesday at 1600 CEST Authorization Panel on this line

You can read a record of call agendas and minutes.

For suggestions to be approved they go through an Editorial review process.

You can keep up to date with Solid specification progress by reading the monthly updates by the Editors which you can also receive by signing up to This Month in Solid.
